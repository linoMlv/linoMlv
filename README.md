## 🚀 Projets principaux

### 🔹 Jaspe  
**Jaspe** est un outil de standardisation et d'architecture que j'ai conçu pour servir de socle à tous mes futurs projets de webapps.  
L'objectif est d'optimiser le développement en posant des bases saines, réutilisables et documentées, garantissant ainsi la maintenabilité des applications sur le long terme.  
👉 [Découvrir la documentation](https://github.com/linoMlv/jaspe)

### 🔹 Abacus  
**Abacus** est une plateforme de gestion comptable claire et modulaire.  
Conçue pour simplifier le suivi financier et l’organisation budgétaire, elle est particulièrement adaptée aux associations et petites structures qui ont besoin d'y voir clair dans leurs données.  
👉 [Voir le projet Abacus](https://github.com/linoMlv/abacus)

### 🔹 Steven-Spy
**Steven-Spy** surveille en continu les sources d'information de votre choix — sites web, flux RSS, APIs — et vous alerte uniquement pour ce qui compte vraiment.
Contrairement aux agrégateurs classiques, STEVEN écrit lui-même ses scripts de scraping en suivant une boucle TDD supervisée par un LLM : vous lui décrivez une source en langage naturel, il code, teste, corrige, déploie. Ensuite il exécute périodiquement ses scripts, filtre les doublons sémantiquement (ChromaDB), note l'importance de chaque nouvelle selon vos critères personnels, et décide — alerte immédiate ou attente du prochain récap.<br/>
👉 [Voir le projet Abacus](https://github.com/linoMlv/steven-spy)
